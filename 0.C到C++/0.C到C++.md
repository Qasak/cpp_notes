## 引用

 定义引用时一定要将其初始化成引用某个变量。
 初始化后，它就一直引用该变量，不会再引用别
的变量了。
 引用只能引用变量，不能引用常量和表达式。  

### 常引用

不能通过常引用去修改其引用的内容:  

```c++
int n = 100;
const int & r = n;
r = 200; //编译错
n = 300; // 没问题
```

const T & 和T & 是不同的类型!!!

T & 类型的引用或T类型的变量可以用来初始化

const T & 类型的引用。

const T 类型的常变量和const T & 类型的引用则
不能用来初始化T &类型的引用，除非进行**强制类型转换 **
